# Feature Specifications 🚀

랠리보드가 제공하는 주요 기능의 상세 작동 방식입니다.

## 1. 대기 명단 관리 (Waiting List)
- **자율 추가:** 사용자가 `AddPlayerForm`을 통해 개별적으로 플레이어를 추가할 수 있습니다.
- **대량 추가 (Bulk Add):** 콤마(`,`)를 구분자로 하여 `"A 이름, B 이름"` 형식으로 한 번에 여러 명을 추가할 수 있습니다.
- **퀵 배치:** 플레이어 네임택의 등급 영역을 클릭하면 나타나는 드롭다운 레이어를 통해 빈 코트로 즉시 배치가 가능합니다.

## 2. 경기 진행 및 스코어 (Scoreboard)
- **자동 경기 종료:** 점수판에서 점수를 조정하여 세트 승무패를 기록하고 경기를 종료할 수 있습니다.
- **경기 기록:** 종료된 경기는 `Match History` 모달에 시간별로 기록됩니다.

## 3. 데이터 영속성 (Data Persistence)
- **LocalStorage 연동:** 모든 상태는 `rally-board-storage` 키로 로컬 스토리지에 자동 저장됩니다.
- **유지 정책:** 사용자가 직접 삭제하지 않는 한 데이터는 보존되며, 마지막 업데이트로부터 **7일(일주일)**이 지나면 자동으로 초기화됩니다.

## 4. 토너먼트 및 타이머
- **이벤트 실행:** 토너먼트 시작 버튼을 누르면 전체 이벤트 타이머가 작동합니다.
- **카운트다운:** 경기 시작 전 시각적인 카운트다운 오버레이를 제공합니다.

## 5. 테마 시스템 (Classic & Retro)
### Classic Mode
- 현대적인 미니멀리즘 디자인
- 부드러운 애니메이션과 그림자 효과
- 직관적인 아이콘 레이아웃

### Retro Mode (NES Style)
- 8-bit 도트 스타일 인터페이스
- `nes.css` 라이브러리 활용
- 픽셀 완벽 정렬(Pixel Perfect) 및 초소형 폰트(6px) 적용
- 고전 게임의 '자석 보드' 느낌 구현

## 6. 인증 및 권한 관리 (Authentication & Approval) [NEW]
- **구글 OAuth 로그인:** 모든 사용자는 서비스를 이용하기 전 구글 계정으로 로그인해야 합니다.
- **관리자 승인 시스템:** 첫 로그인 시 '승인 대기' 상태로 등록되며, 관리자가 `/admin` 페이지에서 승인해야만 점수판 접근이 가능합니다.
- **보안 세션:** `Auth.js (v5)`와 `Supabase Auth`를 연계하여 안전한 세션 관리를 수행합니다.
